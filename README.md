### spring-boot-cloud-module
spring-boot+spring-cloud+maven-module 一个 maven多模块的微服务架构模版。

新手上路的绝佳模版，只有必要的配置文件和核心jar包，没有多余的其他增量配置，所有的配置和注解都有做解释的注解描述，方便新手一眼就懂，下载后基本上可以直接基于demo开发自己的项目
